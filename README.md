# CheckInAmino

برنامج تسجيل دخول يومي في الامينو 

المتطلبات ل التثبيت 

## for install 
```
git clone https://github.com/kira-xc/CheckInAmino
apt install python
pip install Amino.py
```
## for use
```
cd CheckInAmino
python checkin.py
```
